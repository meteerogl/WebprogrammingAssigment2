def anasayfadaki_linkler(sayfa_numarası):
	#ilk sayfadaki siteleri getitiyor
	url = "http://nazarca.com/page"+str(sayfa_numarası)
	response = requests.get(url)
	içerik = response.content
	soup = BeautifulSoup(içerik,"html.parser")
	linkler = []
	for link in soup.find_all("div",{"class":"mh-loop-thumb"},"a"):
    	#print(link.a.get("href"))
    	linkler.append(link.a.get("href"))
	return linkler

This method creates the url of that page according to the page number it gets.First, we have a request from the page using the get () method, and we now have a Response object.Then we get the contents of the contents object.

After all, we are parsing the content using the BeautifulSoup () module, which is used to handle HTML and XML files. And we apply the find_all method to the soup object we are parsing and we take the links with class a name, mh-loop-tumb with a tag and we add them to our link [] array with the append () method.

Bu örneğimiz ile web sitesinden ilk olarak içeriklerin linklerini çekip; bu linkleri bir dizi elemanına eklemeyi öğrendik.
